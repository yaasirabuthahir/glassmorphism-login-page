✨ Glassmorphism Login Page

A modern and stylish login page designed using the glassmorphism UI trend. This project focuses on creating a clean, transparent interface with a blur effect background.

🔹 Features:
- Glassmorphism UI design
- Smooth input animations
- Fully responsive layout
- Minimal and modern look

🛠️ Built With:
- HTML5
- CSS3 (Backdrop Filter, Flexbox)
